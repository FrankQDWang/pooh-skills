# Shared Reporting Style

Use the user's language for narrative text unless the task explicitly requires another language.

The human report must stay decision-first:

- say what is wrong
- say why it matters
- say what should happen next

The agent brief must stay handoff-friendly:

- keep it short
- prefer decisions over tutorials
- describe the target shape and validation gate

Do not pad either output with generic encouragement, command dumps, or duplicated raw logs.
